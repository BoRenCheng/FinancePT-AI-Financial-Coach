**FinancePT** 是一套針對數位原生世代設計的智慧理財輔助系統，參與 **教育部 115 年度 U-start 創新創業計畫**。本專案透過輕量化 AI 技術，提供即時、低門檻且具溫度的金融決策輔助。

**即刻體驗**: [FinancePT Web MVP](https://smart-spend-buddy-36.lovable.app/)

---

## 核心動機
在資訊過載的金融市場中，大眾常面臨「選卡困難」與「理財門檻高」的痛點。FinancePT 透過 **SLM (Small Language Model)** 將複雜的金融數據轉化為直觀的建議，讓理財成為一種「對話式」的簡單習慣 。

---

## 技術亮點 

### 1. 多模態識別與語音記帳 (Multimodal Interaction)
* **語音轉換語意 (Speech-to-Intent)**：支援自然語言口述記帳，系統可自動提取類別、金額與時間 。
* **多維度輸入**：整合影像與文字識別，降低使用者手動輸入的負擔。

### 2. 本地端小型語言模型 (SLM)
* **隱私安全**：專注於輕量化模型推論，減少敏感財務數據上傳至雲端大型 API 的需求 。
* **可解釋性決策**：針對信用卡媒合與消費分析提供具備邏輯基礎的對話建議，降低 AI 幻覺。

### 3. 智慧金融策略引擎 (Financial Strategy)
* **動態選卡邏輯**：根據使用者消費習慣與當前卡片權益，即時推薦最佳支付路徑 。
* **理財素養教育**：將金融知識無縫嵌入對話，提升使用者長期的理財能力。
